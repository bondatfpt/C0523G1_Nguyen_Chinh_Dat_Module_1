Cú pháp clone một repository về máy tính của mình 1. Đi đến trang https://github.com/bondatfpt/C0523G1_Nguyen_Chinh_Dat_Module_1 2. Nhấn vào nút "Clone or download" rồi copy đường dẫn của Repository 3. Mở màn hình Terminal trên máy tính của mình, di chuyển đến thư mục mình muốn chứa các repository (sử dụng lệnh cd). 4. Dùng lệnh git clone để sao chép repository về máy của mình: git clone https://github.com/bondatfpt/C0523G1_Nguyen_Chinh_Dat_Module_1
